# Witness reference voices

These six short reference clips are local development assets generated with the Windows desktop speech synthesizer installed on this workstation. They are suitable for local MiniCPM-o voice-cloning tests and are assigned deterministically to fictional witnesses.

Before distributing this project, replace them with recordings whose redistribution and voice-consent terms are explicitly documented. Keep the `voice_01.wav` through `voice_06.wav` filenames or update `voices.json` and the game assignment logic together.
